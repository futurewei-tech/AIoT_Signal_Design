# AIoT_Signal_Design

# Description
This repository contains the description and matlab code for implementing Method M2-1-1 of CP handling in AIoT R2D link transmission
The description is in the word file: Method M2-1-1 Description.docx.
The matlab code is in the zip file: OOK_gen.zip

#Installation
Unzip file into a matlab work directory. Run the code using the matlab command line. 
Matlab software is required.
